# Training Planner

**Автор:** Бесов Дмитрий Константинович

## Описание
Программа предназначена для планирования тренировок. Вы можете добавлять тренировки с указанием даты, типа и длительности. Встроенная фильтрация позволяет быстро находить нужные записи. Все данные сохраняются в JSON-файл.

## Инструкции по использованию
1. Запустите приложение.
2. Введите дату в формате ДД.ММ.ГГГГ.
3. Выберите тип тренировки из списка или введите вручную.
4. Укажите длительность в минутах.
5. Нажмите «Добавить тренировку».
6. Для фильтрации используйте фильтры по типу и дате.
7. Все данные автоматически сохраняются и загружаются при запуске.

## Возможные тесты
- Добавление нескольких тренировок и проверка отображения.
- Проверка корректности валидации даты и длительности.
- Использование фильтров и сброс фильтров.
