# Random Quote Generator

### Автор
Бесов Дмитрий Константович

### Описание
Это простое GUI-приложение на Python, которое генерирует случайные цитаты из заданного списка. Пользователи могут фильтровать цитаты по авторам и темам, а также сохранять историю сгенерированных цитат.

### Используемые технологии
- Python
- Tkinter для GUI
- JSON для хранения истории
- Git для контроля версий

### Пример использования
1. Запустите программу.
2. Нажмите «Сгенерировать цитату».
3. Посмотрите на появившуюся цитату и добавьте её в историю.
4. Используйте поля фильтрации для поиска цитат по автору и теме.
5. История сохраняется автоматически в `history.json`.
